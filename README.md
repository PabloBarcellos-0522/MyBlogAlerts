## üöÄ What it is

**MyBlogAlerts** is a Python project with automated scraping that:

- Detects new academic posts from disciplines
- Securely stores data in a PostgreSQL database
- Notifies via WhatsApp whenever something new is published

> Complete documentation available at [`docs/index.adoc`](docs/index.adoc)

---

## üì¶ Requirements

- Python 3.12+
- PostgreSQL 13+
- Green API account and key (WhatsApp) or compatible API
- ID of a WhatsApp group that will receive messages (it is possible to generate/view your ID on the Green API website)
- Variables configured in [`.env`](.env.template)

## ‚öôÔ∏è How to use

### 1. Clone the repository

```bash
git clone https://github.com/PabloBarcellos-0522/MyBlogAlerts.git
cd MyBlogAlerts
```

### 2. Configure the environment

- #### Linux
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
- #### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

- Create a .env file in the root with:

```env
DATABASE_URL = postgresql://user:password@host:port/db
API_URL = your_greenAPI_key
GROUP_ID = whatsapp_group_id
BLOG_URL = Main URL of the student portal, EX: (https://aluno.uvv.br/)
WEB_SCRAPER_SECRET_KEY = Key for data encryption (preferably generated by the cryptography lib)
```

### 3. Run the project

```bash
python -m src.interface.Cli
```

---

Done, now you will receive your notices directly in your WhatsApp group.
