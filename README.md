## üöÄ What it is

**MyBlogAlerts** is a Python API with automated scraping that:

- Detects new academic posts from disciplines
- Securely stores data in a PostgreSQL database
- Notifies via WhatsApp whenever something new is published
- Receives and responds to commands sent by users on WhatsApp

> Complete documentation available at [`docs/index.adoc`](docs/index.adoc)

## ‚öôÔ∏è How to use

### 1. Clone the repository

```bash
git clone https://github.com/PabloBarcellos-0522/MyBlogAlerts.git
cd MyBlogAlerts
```

### 2. Configure the environment

- #### Linux
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
- #### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

- Create a .env file in the root with:

```env
DATABASE_URL = postgresql://user:password@host:port/db
API_URL = whatsapp_api_url
API_KEY = whatsapp_api_key
GROUP_ID = whatsapp_group_id (Optional)
BLOG_URL = Main URL of the student portal, EX: (https://aluno.uvv.br/)
WEB_SCRAPER_SECRET_KEY = Key for data encryption (preferably generated by the cryptography lib)
ACESS_TOKEN = MyBlogAlerts
REGISTER_PAGE = url_of_the_registration_page (base to be added to the CORS policy)
REGISTER_PAGE_URL = complete_url_of_the_page
```

### 3. Run the project

```bash
uvicorn src.interface.api:app --reload
```

---

Done, now you will receive your notices directly in your WhatsApp group.

You can also use Local Tunnel if you want to host the API locally:
```bash
lt --port {API_Port}
```
